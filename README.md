# Vue HackerNews

A Hacker News clone with tests, written for the book [Testing Vue.js Applications](https://www.manning.com/books/testing-vuejs-applications)

## Test

Run full suite:

```
npm run test
```

Unit tests:
```
npm run test:unit
```

Integration tests:
```
npm run test:integration
```

E2E tests:
```
npm run test:e2e
```

## Development

Run the development server:

```
npm run serve
```
